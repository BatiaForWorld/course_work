# Приложение для анализа банковских операций

## Описание

Приложение предназначено для анализа транзакций из Excel-файла, формирования отчетов и предоставления сервисов для анализа расходов, кешбэка и других финансовых показателей.


## Структура проекта

- `src/` — исходный код приложения
  - `views.py` — функции для формирования JSON-ответов веб-страниц
  - `services.py` — сервисы для анализа транзакций
  - `reports.py` — отчёты по тратам
  - `utils.py` — вспомогательные функции
- `tests/` — тесты
  - `test_views.py` — тесты для представлений (views)
  - `test_services.py` — тесты для сервисов
  - `test_reports.py` — тесты для отчётов
- `data/` — исходные данные (Excel)
- `user_settings.json` — пользовательские настройки
- `.env` — переменные окружения (ключи API)

## Запуск приложения

Для корректного запуска main.py используйте одну из команд:

Запуск через модуль:
```
poetry run python -m src.main
```

Или с указанием PYTHONPATH:
```
PYTHONPATH=./ poetry run python src/main.py
```

## Запуск тестов

Запуск всех тестов:
```
pytest
```
Запуск тестов для представлений:
```
pytest tests/test_views.py
```
Запуск тестов для сервисов:
```
pytest tests/test_services.py
```
## Запуск тестов с покрытием

## Назначение модулей и функций

### src/utils.py
- Вспомогательные функции для работы с данными и датами.
- `read_excel_data(filepath: str) -> pd.DataFrame`: загружает транзакции из Excel-файла.
- `get_period_dates(date_str: str, period: str = "M") -> tuple`: возвращает диапазон дат для анализа.

### src/reports.py
- Формирование аналитических отчетов по транзакциям.
- `spending_by_category(transactions, category, date)`: траты по категории за 3 месяца.
- `spending_by_weekday(transactions, date)`: средние траты по дням недели за 3 месяца.
- `spending_by_workday(transactions, date)`: средние траты в рабочий/выходной день за 3 месяца.
- Декоратор `save_report_to_file`: сохраняет результат отчёта в JSON-файл.

### src/main.py
- Точка входа для запуска приложения и демонстрации отчётов.
- `main()`: основной метод, который запускает приложение и демонстрирует отчёты.
- Используйте команду в терминале для запуска:
```bash
  python -m src.main
```
Отчёты будут сохранены в папке `report_spending` в формате JSON.
### tests/test_reports.py
- Тесты для всех функций отчётов, используют фикстуры и monkeypatch.

### data/operations.xlsx
- Excel-файл с транзакциями для анализа.

### user_settings.json
- Настройки пользователя: список валют и акций для анализа.

### .env, .env_template
- Переменные окружения и шаблон для ключей API.

### .flake8, pyproject.toml
- Конфигурация стиля, форматирования и типизации.
